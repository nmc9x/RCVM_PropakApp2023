# THÔNG TIN HỆ THỐNG
 Phần mềm cho phép điều khiển nhiều hệ thống.

# Yêu cầu ứng dụng
 * Hệ điều hành:
  - Windows 10 hoặc cao hơn
  
 * Yêu cầu phần mềm:
  - .Net Framework 4.5 trở lên.
  - Tắt các chương trình virus.

* Phiên bảng 1.0.0 (18 Tháng 09, 2023)
  - Phiên bảng dùng thử có các tính năng:
	+ Phân phối Online/Offline
	+ Đồng bộ sản phẩm lên máy chủ
	+ Lịch sử thao tác
	+ Thông tin phần mềm
